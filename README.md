## **💼 接口地址**  
`https://pay.obsdx.pw/?key=秘钥&money=1`

---

## **📋 参数说明**  

| 参数  | 必填 | 说明                |
|-------|------|-------------------|
| key   | 是   | 你的 API 密钥      |
| money | 是   | 支付金额           |

---

## **⚠️ 注意事项**  
- **支付成功后，机器人将自动发送相关日志。**  
- **支付成功的金额会自动增加至狗盾钱包余额。**  
- **最终余额需在狗盾钱包中申请提现。**  
- **每次请求需间隔至少3秒，直到支付成功为止。**  
- **请确保提供正确的 API 密钥及支付金额。**
